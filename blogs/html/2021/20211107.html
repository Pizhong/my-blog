    # 一、初识canvas

    1、什么是canvas?

    <canvas> 是 HTML5 新增的，一个可以使用脚本(通常为 JavaScript) 在其中绘制图像的 HTML 元素。它可以用来制作照片集或者制作简单(也不是那么简单)的动画，甚至可以进行实时视频处理和渲染。

    2、canvas三要素

    id：作为唯一的标识

    width：画布内容宽度的像素大小

    height：画布内容高度的像素大小

    注意：如果不给 <canvas> 设置 widht、height 属性时，则默认 width为300、height 为 150，单位都是 px。也可以使用 css 属性来设置宽高，但是如宽高属性和初始比例不一致，他会出现扭曲。所以，建议永远不要使用 css 属性来设置 <canvas> 的宽高。

    <canvas id="canvas" width="600" height="600"></canvas>
    1

    3、替换内容

    由于某些较老的浏览器（尤其是 IE9 之前的 IE 浏览器）或者浏览器不支持 HTML 元素 <canvas>，在这些浏览器上你应该总是能展示替代内容。

    支持 <canvas> 的浏览器会只渲染 <canvas> 标签，而忽略其中的替代内容。不支持 <canvas> 的浏览器则 会直接渲染替代内容。

     <canvas id="canvas" width="600" height="600">
    你的浏览器不支持 canvas，请升级你的浏览器。
 </canvas>

    # 二、canvas基本使用

    注意：canvas仅仅只是一个画布，绘制需要配合js来进行

    1、创建画布

      <canvas id="canvas1" width="600" height="600"></canvas>
    1

    2、绘制

     // 1、找到画布对象
var canvas1 = document.querySelector('#canvas1')
// console.log(canvas1)
// 2、上下文对象（画笔）
var ctx = canvas1.getContext('2d')
// console.log(ctx)
// 3、绘制路径
ctx.rect(50, 50, 300, 300)
// 4、填充
ctx.fillStyle = '#0f87f0'
ctx.fill()
// 5、描边，渲染路径
ctx.lineWidth = 20
ctx.strokeStyle = '#0ff0a5'
ctx.stroke()

    # 三、canvas的属性及方法

    1、颜色、样式和阴影

    属性 描述
    fillStyle (opens new window) 设置或返回用于填充绘画的颜色、渐变或模式。
    strokeStyle (opens new window) 设置或返回用于笔触的颜色、渐变或模式。
    shadowColor (opens new window) 设置或返回用于阴影的颜色。
    shadowBlur (opens new window) 设置或返回用于阴影的模糊级别。
    shadowOffsetX (opens new window) 设置或返回阴影与形状的水平距离。
    shadowOffsetY (opens new window) 设置或返回阴影与形状的垂直距离。
    方法 描述
    createLinearGradient() (opens new window) 创建线性渐变（用在画布内容上）。
    createPattern() (opens new window) 在指定的方向上重复指定的元素。
    createRadialGradient() (opens new window) 创建放射状/环形的渐变（用在画布内容上）。
    addColorStop() (opens new window) 规定渐变对象中的颜色和停止位置。

    2、线条样式

    属性 描述
    lineCap (opens new window) 设置或返回线条的结束端点样式。
    lineJoin (opens new window) 设置或返回两条线相交时，所创建的拐角类型。
    lineWidth (opens new window) 设置或返回当前的线条宽度。
    miterLimit (opens new window) 设置或返回最大斜接长度。

    3、矩形

    方法 描述
    rect() (opens new window) 创建矩形。
    fillRect() (opens new window) 绘制"被填充"的矩形。
    strokeRect() (opens new window) 绘制矩形（无填充）。
    clearRect() (opens new window) 在给定的矩形内清除指定的像素。

    4、路径

    方法 描述
    fill() (opens new window) 填充当前绘图（路径）。
    stroke() (opens new window) 绘制已定义的路径。
    beginPath() (opens new window) 起始一条路径，或重置当前路径。
    moveTo() (opens new window) 把路径移动到画布中的指定点，不创建线条。
    closePath() (opens new window) 创建从当前点回到起始点的路径。
    lineTo() (opens new window) 添加一个新点，然后在画布中创建从该点到最后指定点的线条。
    clip() (opens new window) 从原始画布剪切任意形状和尺寸的区域。
    quadraticCurveTo() (opens new window) 创建二次贝塞尔曲线。
    bezierCurveTo() (opens new window) 创建三次贝塞尔曲线。
    arc() (opens new window) 创建弧/曲线（用于创建圆形或部分圆）。
    arcTo() (opens new window) 创建两切线之间的弧/曲线。
    isPointInPath() (opens new window) 如果指定的点位于当前路径中，则返回 true，否则返回 false。

    5、转换

    方法 描述
    scale() (opens new window) 缩放当前绘图至更大或更小。
    rotate() (opens new window) 旋转当前绘图。
    translate() (opens new window) 重新映射画布上的 (0,0) 位置。
    transform() (opens new window) 替换绘图的当前转换矩阵。
    setTransform() (opens new window) 将当前转换重置为单位矩阵。然后运行 transform()。

    6、文本

    属性 描述
    font (opens new window) 设置或返回文本内容的当前字体属性。
    textAlign (opens new window) 设置或返回文本内容的当前对齐方式。
    textBaseline (opens new window) 设置或返回在绘制文本时使用的当前文本基线。
    方法 描述
    fillText() (opens new window) 在画布上绘制"被填充的"文本。
    strokeText() (opens new window) 在画布上绘制文本（无填充）。
    measureText() (opens new window) 返回包含指定文本宽度的对象。

    7、图像绘制

    方法 描述
    drawImage() (opens new window) 向画布上绘制图像、画布或视频。

    8、像素操作

    属性 描述
    width (opens new window) 返回 ImageData 对象的宽度。
    height (opens new window) 返回 ImageData 对象的高度。
    data (opens new window) 返回一个对象，其包含指定的 ImageData 对象的图像数据。
    方法 描述
    createImageData() (opens new window) 创建新的、空白的 ImageData 对象。
    getImageData() (opens new window) 返回 ImageData 对象，该对象为画布上指定的矩形复制像素数据。
    putImageData() (opens new window) 把图像数据（从指定的 ImageData 对象）放回画布上。

    9、合成

    属性 描述
    globalAlpha (opens new window) 设置或返回绘图的当前 alpha 或透明值。
    globalCompositeOperation (opens new window) 设置或返回新图像如何绘制到已有的图像上。

    10、其他

    方法 描述
    save() 保存当前环境的状态。
    restore() 返回之前保存过的路径状态和属性。
    createEvent()
    getContext()
    toDataURL()

    # 四、canvas绘制线段

    1、基本步骤

    1. 创建路径起始点
    2. 调用绘制方法去绘制出路径
    3. 把路径封闭
    4. 一旦路径生成，通过描边或填充路径区域来渲染图形。

    2、需要用到的方法

    1. beginPath()

      新建一条路径，路径一旦创建成功，图形绘制命令被指向到路径上生成路径

    2. moveTo(x, y)

      把画笔移动到指定的坐标(x, y)。相当于设置路径的起始点坐标。

    3. closePath()

      闭合路径之后，图形绘制命令又重新指向到上下文中

    4. stroke()

      通过线条来绘制图形轮廓

    5. fill()

      通过填充路径的内容区域生成实心的图形

    3、例子

    <canvas id="tutorial" width="600" height="600"></canvas>
    1
    var tutorial = document.querySelector('#tutorial')
var ctx = tutorial.getContext('2d')

// 新建一条路径
ctx.beginPath()
// 设置绘制的起始点
ctx.moveTo(50, 50)
// 设置经过某个位置
ctx.lineTo(50, 300)
//闭合路径。会拉一条从当前点到path起始点的直线。如果当前点与起始点重合，则什么都不做
ctx.closePath()
// 绘制路径
ctx.lineWidth = 20 // 线段宽度
ctx.strokeStyle = '#0ff0a5' // 线段颜色
ctx.stroke()

    # 五、canvas绘制圆

    1、方法1

    arc(x, y, r, startAngle, endAngle, anticlockwise): 以(x, y) 为圆心，以r 为半径，从 startAngle 弧度开始到endAngle弧度结束。anticlosewise 是布尔值，true 表示逆时针，false 表示顺时针(默认是顺时针)。

    例子：

    <canvas id="circle" width="600" height="600"></canvas>
    1
    var circle = document.querySelector('#circle')
var ctx = circle.getContext('2d')
ctx.arc(300, 300, 100, 0, Math.PI, false)
ctx.stroke()

    2
    3

    2、方法2

    arcTo(x1, y1, x2, y2, radius): 根据给定的控制点和半径画一段圆弧，最后再以直线连接两个控制点。

    例子：

    <canvas id="circle" width="600" height="600"></canvas>
    1
     var circle = document.querySelector('#circle')
 var ctx = circle.getContext('2d')
 ctx.beginPath()
 ctx.moveTo(50, 50)
 //参数1、2：控制点1坐标   参数3、4：控制点2坐标  参数4：圆弧半径
 ctx.arcTo(200, 50, 200, 200, 100)
 ctx.lineTo(200, 200)
 ctx.stroke()

    # 六、绘制文本

    1、fillText(text, x, y [, maxWidth]) 在指定的 (x,y) 位置填充指定的文本，绘制的最大宽度是可选的。

    2、strokeText(text, x, y [, maxWidth]) 在指定的 (x,y) 位置绘制文本边框，绘制的最大宽度是可选的。

    例子：

    <canvas id="text" width="600" height="600"></canvas>
    1
     var text = document.querySelector('#text')
 var ctx = text.getContext('2d')
 ctx.font = "100px sans-serif"
 ctx.fillText("Hello World", 10, 100)
 ctx.strokeText("Hello World", 10, 200)
