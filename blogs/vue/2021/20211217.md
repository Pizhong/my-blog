## `router.js`
``` javascript
export default {
    path: '*',
    name: '404',
    component: () => import('../views/404.vue')
}
```

## 注意

由于路由是从上到下执行的，所以404路由配置要放到所有路由的配置信息的最后，否则会其他路由path匹配造成影响。
